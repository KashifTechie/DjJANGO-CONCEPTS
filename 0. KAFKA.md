# KAFKA

- Apache Kafka is an open-source, fault-tolerant, horizontally scalable distributed event streaming platform used to publish, store, and process real-time data streams reliably and at scale.

1️⃣ Apache Kafka

It is an open-source software system.

Originally developed by LinkedIn.

Now maintained by the Apache Software Foundation.

Used widely in large-scale distributed systems.

2️⃣ Distributed

Kafka runs on multiple servers (brokers) instead of a single machine.

These servers work together as a cluster.

This allows it to handle large amounts of data and traffic.

3️⃣ Event Streaming Platform

Kafka works with events.

An event = something that happened (e.g., user logged in, payment completed, order placed).

“Streaming” means events are processed continuously in real time, not in batches.

Kafka allows:

Publishing events

Storing events

Processing events

4️⃣ Publish

Applications (called Producers) send data to Kafka.

This data is written to topics.

Example: A payment service publishes "payment_success" events.

5️⃣ Store

Kafka stores events on disk.

Data is stored in an ordered, immutable log.

Events can be retained for:

Hours

Days

Weeks

Forever (based on configuration)

6️⃣ Process

Other applications (called Consumers) read events from Kafka.

They can:

Process them

Transform them

Trigger actions

Kafka Streams or external systems can process data in real time.

7️⃣ Fault-Tolerant

Kafka replicates data across multiple brokers.

If one server crashes, data is not lost.

Another broker automatically takes over.

Ensures high availability.

8️⃣ Horizontally Scalable

You can add more servers to the cluster.

This increases:

Storage capacity

Processing power

Throughput

No need to upgrade a single machine (vertical scaling).

9️⃣ Reliable

Kafka ensures:

No message loss (with proper configuration)

Ordered message delivery within partitions

Durable storage

🔟 Used for Real-Time Data Streams

Kafka is commonly used for:

Log aggregation

Event-driven architectures

Real-time analytics

Microservices communication

Fraud detection systems

Monitoring systems
