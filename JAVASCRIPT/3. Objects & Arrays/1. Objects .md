# Objects

---

# 🔹 1. What is an Object?

A collection of **key-value pairs** used to store related data and behavior together.

```javascript
let person = {
    name: "Kashif",
    age: 22,
    city: "Mumbai"
};
```

```
key   → name, age, city
value → "Kashif", 22, "Mumbai"
```

---

# 🔹 2. Creating an Object

---

### ➤ Object Literal — Most Common

```javascript
let person = {
    name: "Kashif",
    age: 22,
    city: "Mumbai"
};
```

---

### ➤ Empty Object — Add Later

```javascript
let person = {};

person.name = "Kashif";
person.age  = 22;

console.log(person); // { name: "Kashif", age: 22 }
```

---

### ➤ Using `new Object()`

```javascript
let person = new Object();

person.name = "Kashif";
person.age  = 22;

console.log(person); // { name: "Kashif", age: 22 }
```

---

# 🔹 3. Accessing Values

---

### ➤ Dot Notation

```javascript
let person = { name: "Kashif", age: 22 };

console.log(person.name); // Kashif
console.log(person.age);  // 22
```

---

### ➤ Bracket Notation

```javascript
console.log(person["name"]); // Kashif
console.log(person["age"]);  // 22
```

---

### ➤ When to Use Bracket Notation

```javascript
// key stored in a variable
let key = "name";
console.log(person[key]); // Kashif

// key has spaces or special characters
let car = { "brand name": "Toyota" };
console.log(car["brand name"]); // Toyota
```

---

# 🔹 4. Setting / Updating Values

---

### ➤ Update Existing Key

```javascript
let person = { name: "Kashif", age: 22 };

person.name = "Ali";
person.age  = 25;

console.log(person); // { name: "Ali", age: 25 }
```

---

### ➤ Add New Key

```javascript
person.city    = "Mumbai";
person.country = "India";

console.log(person);
// { name: "Ali", age: 25, city: "Mumbai", country: "India" }
```

---

# 🔹 5. Deleting a Key

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

delete person.city;

console.log(person); // { name: "Kashif", age: 22 }
```

---

# 🔹 6. Checking if a Key Exists

```javascript
let person = { name: "Kashif", age: 22 };

console.log("name" in person); // true
console.log("city" in person); // false
```

---

# 🔹 7. Looping Over an Object

### ➤ `for...in` — Loop Over Keys

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

for (let key in person) {
    console.log(key, "→", person[key]);
}
// name → Kashif
// age  → 22
// city → Mumbai
```

---

# 🔹 8. Nested Objects

---

### ➤ Object Inside Object

```javascript
let person = {
    name: "Kashif",
    age: 22,
    address: {
        city: "Mumbai",
        country: "India"
    }
};

console.log(person.address.city);    // Mumbai
console.log(person.address.country); // India
```

---

### ➤ Update Nested Value

```javascript
person.address.city = "Delhi";

console.log(person.address.city); // Delhi
```

---

# 🔹 9. Object with Methods

---

### ➤ Adding a Method

```javascript
let person = {
    name: "Kashif",
    age: 22,

    greet() {
        console.log("Hello, I am " + this.name);
    },

    isAdult() {
        return this.age >= 18;
    }
};

person.greet();                // Hello, I am Kashif
console.log(person.isAdult()); // true
```

---

### ➤ `this` Keyword Inside Object

```javascript
let person = {
    name: "Kashif",
    age: 22,

    getDetails() {
        console.log("Name: " + this.name); // this → refers to person
        console.log("Age: "  + this.age);
    }
};

person.getDetails();
// Name: Kashif
// Age: 22
```

# 🔹 10. Destructuring

Extract values from object into variables cleanly.

---

### ➤ Basic Destructuring

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

let { name, age, city } = person;

console.log(name); // Kashif
console.log(age);  // 22
console.log(city); // Mumbai
```

---

### ➤ With Default Values

```javascript
let person = { name: "Kashif", age: 22 };

let { name, age, city = "Unknown" } = person;

console.log(city); // Unknown ← key didn't exist, default used
```

---

### ➤ Rename While Destructuring

```javascript
let person = { name: "Kashif", age: 22 };

let { name: fullName, age: years } = person;

console.log(fullName); // Kashif
console.log(years);    // 22
```

---

# 🔹 11. Spread and Merge

---

### ➤ Merge Two Objects

```javascript
let personal = { name: "Kashif", age: 22 };
let contact  = { city: "Mumbai", phone: "1234" };

let merged = { ...personal, ...contact };

console.log(merged);
// { name: "Kashif", age: 22, city: "Mumbai", phone: "1234" }
```

---

### ➤ Override While Merging

```javascript
let defaults     = { theme: "light", language: "en", fontSize: 14 };
let userSettings = { theme: "dark" };

let settings = { ...defaults, ...userSettings };

console.log(settings);
// { theme: "dark", language: "en", fontSize: 14 }
```

# 🔹 12. Built-in Object Methods

| Method | What it does |
|--------|-------------|
| `Object.keys()` | Returns array of all keys |
| `Object.values()` | Returns array of all values |
| `Object.entries()` | Returns array of [key, value] pairs |
| `Object.fromEntries()` | Converts array of [key, value] pairs back to object |
| `Object.assign()` | Copies properties into target object |
| `Object.freeze()` | Makes object immutable — no add, update, delete |
| `Object.seal()` | Allows update but no add or delete |
| `Object.isFrozen()` | Checks if object is frozen |
| `Object.isSealed()` | Checks if object is sealed |
| `Object.create()` | Creates new object with given prototype |
| `Object.hasOwn()` | Checks if key exists on object |
| `hasOwnProperty()` | Checks if key exists on object (older way) |

---

#### `Object.keys()`

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

console.log(Object.keys(person));
// ["name", "age", "city"]
```

---

#### `Object.values()`

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

console.log(Object.values(person));
// ["Kashif", 22, "Mumbai"]
```

---

#### `Object.entries()`

```javascript
let person = { name: "Kashif", age: 22, city: "Mumbai" };

console.log(Object.entries(person));
// [["name", "Kashif"], ["age", 22], ["city", "Mumbai"]]
```

---

#### `Object.fromEntries()`

Opposite of `Object.entries()` — converts array of pairs back to object.

```javascript
let entries = [["name", "Kashif"], ["age", 22], ["city", "Mumbai"]];

let person = Object.fromEntries(entries);

console.log(person);
// { name: "Kashif", age: 22, city: "Mumbai" }
```

---

#### `Object.assign()`

```javascript
let target = { a: 1 };
let source = { b: 2, c: 3 };

Object.assign(target, source);

console.log(target); // { a: 1, b: 2, c: 3 }
```

---

#### `Object.freeze()`

No **add**, **update**, or **delete** allowed after freeze.

```javascript
let car = { brand: "Toyota", year: 2020 };

Object.freeze(car);

car.brand = "Honda"; // ❌ silently fails
car.color = "Red";   // ❌ silently fails
delete car.year;     // ❌ silently fails

console.log(car); // { brand: "Toyota", year: 2020 } → unchanged
```

---

#### `Object.seal()`

**Update** allowed — but no **add** or **delete**.

```javascript
let car = { brand: "Toyota", year: 2020 };

Object.seal(car);

car.brand = "Honda"; // ✅ update works
car.color = "Red";   // ❌ add fails silently
delete car.year;     // ❌ delete fails silently

console.log(car); // { brand: "Honda", year: 2020 }
```

---

#### `Object.isFrozen()`

```javascript
let car = { brand: "Toyota" };

console.log(Object.isFrozen(car)); // false

Object.freeze(car);

console.log(Object.isFrozen(car)); // true
```

---

#### `Object.isSealed()`

```javascript
let car = { brand: "Toyota" };

console.log(Object.isSealed(car)); // false

Object.seal(car);

console.log(Object.isSealed(car)); // true
```

---

#### `Object.create()`

Creates a new object using an existing object as its prototype.

```javascript
let animal = {
    speak() {
        console.log(this.name + " makes a sound");
    }
};

let dog = Object.create(animal);
dog.name = "Bruno";

dog.speak(); // Bruno makes a sound
```

---

#### `Object.hasOwn()` — Modern Way

```javascript
let person = { name: "Kashif", age: 22 };

console.log(Object.hasOwn(person, "name")); // true
console.log(Object.hasOwn(person, "city")); // false
```

---

#### `hasOwnProperty()` — Older Way

```javascript
let person = { name: "Kashif", age: 22 };

console.log(person.hasOwnProperty("name")); // true
console.log(person.hasOwnProperty("city")); // false
```

# 🔥 Final Summary Table

| Operation | How |
|-----------|-----|
| Create | `let obj = { key: value }` |
| Access | `obj.key` or `obj["key"]` |
| Update | `obj.key = newValue` |
| Add key | `obj.newKey = value` |
| Delete key | `delete obj.key` |
| Check key exists | `"key" in obj` |
| Loop | `for...in` |
| Destructure | `let { key } = obj` |
| Merge | `{ ...obj1, ...obj2 }` |
| Freeze | `Object.freeze(obj)` |
| Seal | `Object.seal(obj)` |

---

# 💡 Key Takeaway

- Use **dot notation** for normal access, **bracket notation** for dynamic keys
- Use **`this`** inside methods to refer to the same object
- Use **`Object.freeze()`** when nothing should change at all
- Use **`Object.seal()`** when updates are allowed but no add or delete
- Use **`Object.keys/values/entries()`** to loop or convert object data
- Use **`Object.fromEntries()`** to convert array of pairs back to object
