# Arrays

---

# 🔹 1. What is an Array?

An **ordered collection of values** stored at numbered indexes starting from 0.

```javascript
let fruits = ["apple", "banana", "mango"];
```

```
index →   0        1         2
value → "apple" "banana"  "mango"
```

---

# 🔹 2. Creating an Array

---

### ➤ Array Literal — Most Common

```javascript
let fruits  = ["apple", "banana", "mango"];
let numbers = [1, 2, 3, 4, 5];
let mixed   = ["Kashif", 22, true, null];   // any type
```

---

### ➤ Empty Array — Add Later

```javascript
let fruits = [];

fruits[0] = "apple";
fruits[1] = "banana";

console.log(fruits); // ["apple", "banana"]
```

---

### ➤ Using `new Array()`

```javascript
let fruits = new Array("apple", "banana", "mango");

console.log(fruits); // ["apple", "banana", "mango"]
```

---

# 🔹 3. Accessing Values

```javascript
let fruits = ["apple", "banana", "mango"];

console.log(fruits[0]); // apple
console.log(fruits[1]); // banana
console.log(fruits[2]); // mango
```

---

### ➤ Access Last Element

```javascript
console.log(fruits[fruits.length - 1]); // mango

// Modern way
console.log(fruits.at(-1)); // mango
console.log(fruits.at(-2)); // banana
```

---

# 🔹 4. Setting / Updating Values

```javascript
let fruits = ["apple", "banana", "mango"];

fruits[1] = "grape";

console.log(fruits); // ["apple", "grape", "mango"]
```

---

# 🔹 5. Checking Length

```javascript
let fruits = ["apple", "banana", "mango"];

console.log(fruits.length); // 3
```

---

# 🔹 6. Looping Over an Array

---

### ➤ `for` Loop

```javascript
let fruits = ["apple", "banana", "mango"];

for (let i = 0; i < fruits.length; i++) {
    console.log(i, fruits[i]);
}
// 0 apple
// 1 banana
// 2 mango
```

---

### ➤ `for...of` Loop

```javascript
for (let fruit of fruits) {
    console.log(fruit);
}
// apple
// banana
// mango
```

# 🔹 7. Nested Arrays (2D Array)

```javascript
let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log(matrix[0][0]); // 1
console.log(matrix[1][2]); // 6
console.log(matrix[2][1]); // 8
```

---

### ➤ Looping Over Nested Array

```javascript
for (let row of matrix) {
    for (let col of row) {
        console.log(col);
    }
}
// 1 2 3 4 5 6 7 8 9
```

# 🔹 8. Destructuring

---

### ➤ Basic Destructuring

```javascript
let fruits = ["apple", "banana", "mango"];

let [first, second, third] = fruits;

console.log(first);  // apple
console.log(second); // banana
console.log(third);  // mango
```

---

### ➤ Skip Elements

```javascript
let [first, , third] = fruits;

console.log(first); // apple
console.log(third); // mango
```

---

### ➤ With Default Values

```javascript
let [a, b, c, d = "grape"] = fruits;

console.log(d); // grape ← index didn't exist, default used
```

---

### ➤ Rest in Destructuring

```javascript
let [first, ...rest] = fruits;

console.log(first); // apple
console.log(rest);  // ["banana", "mango"]
```

| `findIndex()` | Returns index of first match | ❌ No |

## ➤ Adding / Removing Elements

---

#### `push()` — Add to End

```javascript
let arr = [1, 2, 3];

arr.push(4);
arr.push(5, 6); // multiple at once

console.log(arr); // [1, 2, 3, 4, 5, 6]
```

---

#### `pop()` — Remove from End

```javascript
let arr = [1, 2, 3];

let removed = arr.pop();

console.log(removed); // 3
console.log(arr);     // [1, 2]
```

---

#### `unshift()` — Add to Beginning

```javascript
let arr = [1, 2, 3];

arr.unshift(0);
arr.unshift(-2, -1); // multiple at once

console.log(arr); // [-2, -1, 0, 1, 2, 3]
```

---

#### `shift()` — Remove from Beginning

```javascript
let arr = [1, 2, 3];

let removed = arr.shift();

console.log(removed); // 1
console.log(arr);     // [2, 3]
```

---

#### `splice()` — Add / Remove at Any Position

```javascript
// Remove
let arr = [1, 2, 3, 4, 5];

arr.splice(2, 2); // start at index 2, remove 2 elements

console.log(arr); // [1, 2, 5]
```

```javascript
// Add
let arr = [1, 2, 5];

arr.splice(2, 0, 3, 4); // at index 2, remove 0, add 3 and 4

console.log(arr); // [1, 2, 3, 4, 5]
```

```javascript
// Replace
let arr = [1, 2, 99, 4, 5];

arr.splice(2, 1, 3); // at index 2, remove 1, add 3

console.log(arr); // [1, 2, 3, 4, 5]
```

## ➤ Sorting / Ordering

---

#### `sort()` — Sort in Place

```javascript
let names = ["Zara", "Ali", "Kashif"];

names.sort();

console.log(names); // ["Ali", "Kashif", "Zara"]
```

```javascript
// Numbers — must use comparator
let nums = [10, 1, 5, 3];

nums.sort((a, b) => a - b); // ascending
console.log(nums); // [1, 3, 5, 10]

nums.sort((a, b) => b - a); // descending
console.log(nums); // [10, 5, 3, 1]
```

---

#### `reverse()` — Reverse in Place

```javascript
let arr = [1, 2, 3, 4, 5];

arr.reverse();

console.log(arr); // [5, 4, 3, 2, 1]
```

---

#### `fill()` — Fill with a Value

```javascript
let arr = [1, 2, 3, 4, 5];

arr.fill(0);         // fill all with 0
console.log(arr);    // [0, 0, 0, 0, 0]

arr.fill(9, 2, 4);   // fill 9 from index 2 to 4
console.log(arr);    // [0, 0, 9, 9, 0]
```

## ➤ Transforming

---

#### `map()` — Transform Every Element

```javascript
let numbers = [1, 2, 3, 4];

let squared = numbers.map(n => n * n);

console.log(squared); // [1, 4, 9, 16]
console.log(numbers); // [1, 2, 3, 4] → unchanged
```

```javascript
// Real example — extract names from objects
let users = [
    { name: "Kashif", age: 22 },
    { name: "Ali",    age: 17 },
    { name: "Sara",   age: 25 }
];

let names = users.map(user => user.name);

console.log(names); // ["Kashif", "Ali", "Sara"]
```

---

#### `filter()` — Keep Matching Elements

```javascript
let numbers = [1, 2, 3, 4, 5, 6];

let evens = numbers.filter(n => n % 2 === 0);

console.log(evens); // [2, 4, 6]
```

```javascript
// Real example — filter adults
let users = [
    { name: "Kashif", age: 22 },
    { name: "Ali",    age: 17 },
    { name: "Sara",   age: 25 }
];

let adults = users.filter(user => user.age >= 18);

console.log(adults);
// [{ name: "Kashif", age: 22 }, { name: "Sara", age: 25 }]
```

---

#### `reduce()` — Reduce to Single Value

```javascript
let numbers = [1, 2, 3, 4, 5];

let sum = numbers.reduce((acc, curr) => acc + curr, 0);

console.log(sum); // 15
```

```
acc  → accumulator (running total, starts at 0)
curr → current element being processed
```

```javascript
// Real example — total price of cart
let cart = [
    { item: "Laptop", price: 1200 },
    { item: "Phone",  price: 800  },
    { item: "Watch",  price: 200  }
];

let total = cart.reduce((acc, product) => acc + product.price, 0);

console.log(total); // 2200
```

---

#### `flat()` — Flatten Nested Arrays

```javascript
let nested = [1, [2, 3], [4, [5, 6]]];

console.log(nested.flat());    // [1, 2, 3, 4, [5, 6]] → 1 level
console.log(nested.flat(2));   // [1, 2, 3, 4, 5, 6]   → 2 levels
console.log(nested.flat(Infinity)); // flatten all levels
```

---

#### `flatMap()` — Map then Flatten

```javascript
let sentences = ["Hello World", "Good Morning"];

let words = sentences.flatMap(s => s.split(" "));

console.log(words); // ["Hello", "World", "Good", "Morning"]
```

## ➤ Searching

---

#### `find()` — First Element that Matches

```javascript
let users = [
    { id: 1, name: "Ali"    },
    { id: 2, name: "Kashif" },
    { id: 3, name: "Sara"   }
];

let user = users.find(u => u.id === 2);

console.log(user); // { id: 2, name: "Kashif" }
```

---

#### `findIndex()` — Index of First Match

```javascript
let users = [
    { id: 1, name: "Ali"    },
    { id: 2, name: "Kashif" },
    { id: 3, name: "Sara"   }
];

let index = users.findIndex(u => u.id === 2);

console.log(index); // 1
```

---

#### `includes()` — Check if Value Exists

```javascript
let arr = [10, 20, 30];

console.log(arr.includes(20)); // true
console.log(arr.includes(99)); // false
```

---

#### `indexOf()` — Index of Value (-1 if not found)

```javascript
let arr = ["a", "b", "c", "b"];

console.log(arr.indexOf("b"));     // 1 → first occurrence
console.log(arr.lastIndexOf("b")); // 3 → last occurrence
console.log(arr.indexOf("z"));     // -1 → not found
```

## ➤ Checking Conditions

---

#### `some()` — At Least One Passes

```javascript
let numbers = [1, 2, 3, 4, 5];

console.log(numbers.some(n => n > 4)); // true  → 5 passes
console.log(numbers.some(n => n > 9)); // false → none pass
```

---

#### `every()` — All Must Pass

```javascript
let numbers = [2, 4, 6, 8];

console.log(numbers.every(n => n % 2 === 0)); // true  → all even
console.log(numbers.every(n => n > 5));       // false → not all > 5
```

## ➤ Slicing / Joining

---

#### `slice()` — Get Portion (Non-destructive)

```javascript
let arr = [1, 2, 3, 4, 5];

console.log(arr.slice(1, 3)); // [2, 3]  → index 1 to 3 (3 excluded)
console.log(arr.slice(2));    // [3, 4, 5] → from index 2 to end
console.log(arr.slice(-2));   // [4, 5]  → last 2 elements
console.log(arr);             // [1, 2, 3, 4, 5] → unchanged
```

---

#### `concat()` — Merge Arrays

```javascript
let a = [1, 2, 3];
let b = [4, 5, 6];
let c = [7, 8, 9];

let merged = a.concat(b, c);

console.log(merged); // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

---

#### `join()` — Array to String

```javascript
let fruits = ["apple", "banana", "mango"];

console.log(fruits.join());      // apple,banana,mango
console.log(fruits.join(" - ")); // apple - banana - mango
console.log(fruits.join(""));    // applebananamango
```

## ➤ Looping

---

#### `forEach()` — Loop with No Return

```javascript
let fruits = ["apple", "banana", "mango"];

fruits.forEach((fruit, index) => {
    console.log(index, fruit);
});
// 0 apple
// 1 banana
```

## ➤ Utility Methods

---

#### `at()` — Access by Index (Supports Negative)

```javascript
let arr = [10, 20, 30, 40, 50];

console.log(arr.at(0));  // 10 → first
console.log(arr.at(-1)); // 50 → last
console.log(arr.at(-2)); // 40 → second from last
```

---

#### `Array.isArray()` — Check if Value is Array

```javascript
console.log(Array.isArray([1, 2, 3])); // true
console.log(Array.isArray("hello"));   // false
console.log(Array.isArray({ a: 1 }));  // false
```

---

#### `Array.from()` — Create Array from Iterable

```javascript
// From string
console.log(Array.from("hello")); // ["h", "e", "l", "l", "o"]

// From Set
let set = new Set([1, 2, 3]);
console.log(Array.from(set)); // [1, 2, 3]

// With map function
console.log(Array.from([1, 2, 3], n => n * 2)); // [2, 4, 6]
```

## ➤ Chaining Methods

Methods can be **chained** one after another for clean, readable code.

```javascript
let orders = [
    { item: "Laptop", price: 1200, delivered: true  },
    { item: "Phone",  price: 800,  delivered: false },
    { item: "Watch",  price: 200,  delivered: true  },
    { item: "Tablet", price: 600,  delivered: false }
];

let total = orders
    .filter(o => o.delivered)           // keep delivered only
    .map(o => o.price)                  // extract prices
    .reduce((acc, price) => acc + price, 0); // sum them

console.log(total); // 1400
```

# 💡 Key Takeaway

- **Mutating methods** (`push`, `pop`, `splice`, `sort`) change the original array
- **Non-mutating methods** (`map`, `filter`, `reduce`, `slice`) return new arrays
- Use **`map`** to transform, **`filter`** to select, **`reduce`** to accumulate
- Use **`some`** to check if any pass, **`every`** to check if all pass
- Use **`find`** to get element, **`findIndex`** to get its position
- **Chain methods** for clean and readable data processing
