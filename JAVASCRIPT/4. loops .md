# Loops

---

# 🔹 1. `for` Loop

Used when you know **exactly how many times** you want to repeat something.

---

### ➤ Syntax

```javascript
for (initialization; condition; update) {
    // code to repeat
}
```

```
1. initialization → runs once at the start (let i = 0)
2. condition      → checked before every iteration (i < 5)
3. update         → runs after every iteration (i++)
```

---

### ➤ Basic Example

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
// 0
// 1
// 2
// 3
// 4
```

---

### ➤ Looping Over an Array

```javascript
let fruits = ["apple", "banana", "mango"];

for (let i = 0; i < fruits.length; i++) {
    console.log(i, fruits[i]);
}
// 0 apple
// 1 banana
// 2 mango
```

---

### ➤ Looping in Reverse

```javascript
let fruits = ["apple", "banana", "mango"];

for (let i = fruits.length - 1; i >= 0; i--) {
    console.log(fruits[i]);
}
// mango
// banana
// apple
```

---

### ➤ Looping with a Step

```javascript
for (let i = 0; i <= 10; i += 2) {
    console.log(i);
}
// 0 2 4 6 8 10
```

---

### ➤ Nested `for` Loop

```javascript
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(i, j);
    }
}
// 1 1
// 1 2
// 1 3
// 2 1
// ... and so on
```

# 🔹 2. `while` Loop

Used when you **don't know how many times** to loop — runs as long as condition is true.

---

### ➤ Syntax

```javascript
while (condition) {
    // code to repeat
}
```

---

### ➤ Basic Example

```javascript
let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}
// 0
// 1
// 2
// 3
// 4
```

---

### ➤ Real World Example — Wait Until Condition Met

```javascript
let stock = 10;

while (stock > 0) {
    console.log("Items left: " + stock);
    stock--;
}
// Items left: 10
// Items left: 9
// ...
// Items left: 1
```

---

### ➤ `do...while` Loop

Runs the block **at least once**, then checks the condition.

```javascript
let i = 0;

do {
    console.log(i);
    i++;
} while (i < 3);
// 0
// 1
// 2
```

---

### ➤ Difference — `while` vs `do...while`

```javascript
// while → condition checked BEFORE running
let i = 10;

while (i < 5) {
    console.log(i); // ❌ never runs
}

// do...while → condition checked AFTER running
do {
    console.log(i); // ✅ runs once → 10
} while (i < 5);
```

---

### ➤ ⚠️ Infinite Loop — Always Avoid

```javascript
// ❌ BAD — i never changes, runs forever
while (true) {
    console.log("infinite");
}

// ✅ Always make sure the condition eventually becomes false
let i = 0;
while (i < 5) {
    console.log(i);
    i++; // this makes it stop
}
```

# 🔹 3. `for...of` Loop

Used to loop over **values** of any iterable — Array, String, Map, Set.

---

### ➤ Syntax

```javascript
for (let value of iterable) {
    // use value
}
```

---

### ➤ Looping Over an Array

```javascript
let fruits = ["apple", "banana", "mango"];

for (let fruit of fruits) {
    console.log(fruit);
}
// apple
// banana
// mango
```

---

### ➤ Looping Over a String

```javascript
let name = "Kashif";

for (let char of name) {
    console.log(char);
}
// K
// a
// s
// h
// i
// f
```

---

### ➤ Looping Over a Set

```javascript
let set = new Set([1, 2, 3]);

for (let value of set) {
    console.log(value);
}
// 1
// 2
// 3
```

---

### ➤ Looping Over a Map

```javascript
let map = new Map();
map.set("name", "Kashif");
map.set("age", 22);

for (let [key, value] of map) {
    console.log(key, value);
}
// name Kashif
// age 22
```

# 🔹 4. `for...in` Loop

Used to loop over **keys** of an Object.

---

### ➤ Syntax

```javascript
for (let key in object) {
    // use key
}
```

---

### ➤ Looping Over an Object

```javascript
let person = {
    name: "Kashif",
    age: 22,
    city: "Mumbai"
};

for (let key in person) {
    console.log(key, person[key]);
}
// name Kashif
// age 22
// city Mumbai
```

---

### ➤ Accessing Only Keys

```javascript
for (let key in person) {
    console.log(key);
}
// name
// age
// city
```

---

### ➤ Accessing Only Values

```javascript
for (let key in person) {
    console.log(person[key]);
}
// Kashif
// 22
// Mumbai
```

---

### ➤ ⚠️ Don't Use `for...in` on Arrays

```javascript
let fruits = ["apple", "banana", "mango"];

// ❌ BAD — gives indexes as strings, not values
for (let i in fruits) {
    console.log(i); // "0" "1" "2" → string indexes
}

// ✅ GOOD — use for...of for arrays
for (let fruit of fruits) {
    console.log(fruit); // apple banana mango
}
```

# 🔹 5. `break` and `continue`

Used to **control the flow** inside any loop.

---

### ➤ `break` — Exit the loop completely

```javascript
for (let i = 0; i < 5; i++) {
    if (i === 3) {
        break; // stops the loop
    }
    console.log(i);
}
// 0
// 1
// 2
```

---

### ➤ `continue` — Skip current iteration, move to next

```javascript
for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue; // skips 2
    }
    console.log(i);
}
// 0
// 1
// 3
// 4
```

# 🔹 6. Choosing the Right Loop

| Situation | Use |
|-----------|-----|
| Know exact number of iterations | `for` |
| Loop until a condition changes | `while` |
| Must run at least once | `do...while` |
| Loop over values of array / string / map / set | `for...of` |
| Loop over keys of an object | `for...in` |

---

### ➤ Example — Same Array, All Loops

```javascript
let nums = [10, 20, 30];

// for
for (let i = 0; i < nums.length; i++) {
    console.log(nums[i]);
}

// while
let i = 0;
while (i < nums.length) {
    console.log(nums[i]);
    i++;
}

// for...of
for (let num of nums) {
    console.log(num);
}
```

> All three print `10 20 30` — pick based on what fits the situation.

---

# 🔥 Final Summary Table

| Loop | Iterates Over | Gives You | Use When |
|------|--------------|-----------|----------|
| `for` | Anything with index | Index + value | Exact count known |
| `while` | Anything | Manual control | Condition-based |
| `do...while` | Anything | Manual control | Must run at least once |
| `for...of` | Array, String, Map, Set | Value directly | Iterating values |
| `for...in` | Object | Key | Iterating object keys |

---

# 💡 Key Takeaway

- Use **`for`** when count is known
- Use **`while`** when looping until something changes
- Use **`for...of`** for arrays, strings, maps, sets — gives values directly
- Use **`for...in`** for objects — gives keys
- Always use **`break`** / **`continue`** to control loop flow when needed
