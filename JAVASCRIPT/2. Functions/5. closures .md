# Closures

---

# 🔹 What is a Closure?

A **function that remembers variables from its outer scope** even after the outer function has finished.

---

# 🔹 Example 1 — Basic Closure

```javascript
function outer() {
    let name = "Kashif";

    function inner() {
        console.log(name);   // remembers name from outer
    }

    return inner;
}

let fn = outer();   // outer() is done — name should be gone
fn();               // Kashif ← but inner still remembers it
```

```
outer() finished → but inner still holds name = "Kashif"
That is a closure.
```

---

# 🔹 Example 2 — Counter (Most Common Use)

```javascript
function createCounter() {
    let count = 0;             // private variable

    return function() {
        count++;
        console.log(count);
    };
}

let counter = createCounter();

counter(); // 1
counter(); // 2
counter(); // 3
```

```
count is NOT reset on every call
inner function remembers and updates the same count
count cannot be accessed or changed from outside → private
```

---

# 🔹 Example 3 — Function Factory

```javascript
function createMultiplier(factor) {
    return function(number) {
        return number * factor;   // remembers factor
    };
}

let double = createMultiplier(2);
let triple = createMultiplier(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```

```
double → remembers factor = 2
triple → remembers factor = 3
each has its own separate closure
```

---

# 💡 Key Takeaway

- Closure = inner function **remembers** outer variables even after outer is done
- Every closure gets its **own separate copy** of outer variables
- Used for **private variables** and **function factories**
