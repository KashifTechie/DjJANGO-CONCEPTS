# User Defined Functions

---

# 🔹 1. What is a Function?

A **reusable block of code** that runs only when called.

```
Parameter → variable listed in the function definition
Argument  → actual value passed when calling the function
Return    → sends a value back to whoever called the function
```

---

# 🔹 2. Ways to Define a Function

---

### ➤ Function Declaration

```javascript
function greet() {
    console.log("Hello");
}
```

---

### ➤ Function Expression

```javascript
const greet = function() {
    console.log("Hello");
};
```

---

### ➤ Arrow Function

```javascript
const greet = () => {
    console.log("Hello");
};
```

> All three ways work the same for our 4 scenarios below — examples will use **Function Declaration** for clarity.

---

# 🔹 3. Scenario 1 — With Parameters and Return

Function **receives input** and **sends back output**.

---

### ➤ How it Works

```
caller  →  passes arguments
function →  receives as parameters
function →  processes and returns result
caller  →  gets the result back
```

---

### ➤ Basic Example

```javascript
function add(a, b) {
    return a + b;
}

let result = add(5, 3);

console.log(result); // 8
```

---

### ➤ Example — Full Name

```javascript
function getFullName(firstName, lastName) {
    return firstName + " " + lastName;
}

let name = getFullName("Kashif", "Ali");

console.log(name); // Kashif Ali
```

---

### ➤ Example — Check Even or Odd

```javascript
function isEven(number) {
    return number % 2 === 0;
}

console.log(isEven(4));  // true
console.log(isEven(7));  // false
```

---

### ➤ Example — Find Larger Number

```javascript
function findMax(a, b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}

console.log(findMax(10, 20)); // 20
console.log(findMax(55, 30)); // 55
```

---

### ➤ Example — Calculate Area

```javascript
function calculateArea(length, width) {
    let area = length * width;
    return area;
}

let area = calculateArea(5, 10);

console.log(area); // 50
```

---

### ➤ ⚠️ Important — `return` Stops the Function

```javascript
function check(num) {
    if (num < 0) {
        return "Negative"; // exits here if true
    }
    return "Positive";     // only reaches here if num >= 0
}

console.log(check(-5)); // Negative
console.log(check(10)); // Positive
```

---

# 🔹 4. Scenario 2 — With Parameters and No Return

Function **receives input**, does something with it, but **sends nothing back**.

---

### ➤ How it Works

```
caller   →  passes arguments
function →  receives as parameters
function →  does a task (print, update, log)
caller   →  gets nothing back (undefined)
```

---

### ➤ Basic Example

```javascript
function greet(name) {
    console.log("Hello, " + name);
}

greet("Kashif"); // Hello, Kashif
```

---

### ➤ Example — Print Square

```javascript
function printSquare(num) {
    let square = num * num;
    console.log(num + " squared is " + square);
}

printSquare(4); // 4 squared is 16
printSquare(7); // 7 squared is 49
```

---

### ➤ Example — Print All Items in Array

```javascript
function printItems(items) {
    for (let item of items) {
        console.log(item);
    }
}

printItems(["apple", "banana", "mango"]);
// apple
// banana
// mango
```

---

### ➤ Example — Print Person Details

```javascript
function printDetails(name, age, city) {
    console.log("Name: " + name);
    console.log("Age: " + age);
    console.log("City: " + city);
}

printDetails("Kashif", 22, "Mumbai");
// Name: Kashif
// Age: 22
// City: Mumbai
```

---

### ➤ ⚠️ If You Try to Capture the Result

```javascript
function greet(name) {
    console.log("Hello, " + name);
}

let result = greet("Kashif"); // Hello, Kashif

console.log(result); // undefined → nothing was returned
```

# 🔹 5. Scenario 3 — Without Parameters and With Return

Function **takes no input** but **sends back output**.

---

### ➤ How it Works

```
caller   →  calls function with no arguments
function →  does its own work internally
function →  returns result
caller   →  gets the result back
```

---

### ➤ Basic Example

```javascript
function getGreeting() {
    return "Hello, World!";
}

let message = getGreeting();

console.log(message); // Hello, World!
```

---

### ➤ Example — Get Current Year

```javascript
function getCurrentYear() {
    let year = new Date().getFullYear();
    return year;
}

let year = getCurrentYear();

console.log(year); // 2026
```

---

### ➤ Example — Get Random Number

```javascript
function getRandomNumber() {
    return Math.floor(Math.random() * 100);
}

console.log(getRandomNumber()); // e.g. 42
console.log(getRandomNumber()); // e.g. 87
```

---

### ➤ Example — Get App Name

```javascript
function getAppName() {
    return "MyApp v1.0";
}

let appName = getAppName();

console.log(appName); // MyApp v1.0
```

---

# 🔹 6. Scenario 4 — Without Parameters and No Return

Function **takes no input** and **sends nothing back** — just performs a task.

---

### ➤ How it Works

```
caller   →  calls function with no arguments
function →  performs a fixed task (print, log, update)
caller   →  gets nothing back
```

---

### ➤ Basic Example

```javascript
function greet() {
    console.log("Hello, World!");
}

greet(); // Hello, World!
```

---

### ➤ Example — Print Divider

```javascript
function printDivider() {
    console.log("--------------------");
}

printDivider(); // --------------------
```

---

### ➤ Example — Print App Info

```javascript
function printAppInfo() {
    console.log("App: MyApp");
    console.log("Version: 1.0");
    console.log("Author: Kashif");
}

printAppInfo();
// App: MyApp
// Version: 1.0
// Author: Kashif
```

---

### ➤ Example — Show Welcome Message

```javascript
function showWelcome() {
    console.log("====================");
    console.log("  Welcome to MyApp  ");
    console.log("====================");
}

showWelcome();
// ====================
//   Welcome to MyApp
// ====================
```

# 🔹 7. Default Parameters

If no argument is passed, the parameter uses a **default value**.

```javascript
function greet(name = "Guest") {
    return "Hello, " + name;
}

console.log(greet("Kashif")); // Hello, Kashif
console.log(greet());         // Hello, Guest
```

---

# 🔹 8. Rest Parameters

Collect **multiple arguments** into a single array using `...`

```javascript
function addAll(...numbers) {
    let sum = 0;
    for (let num of numbers) {
        sum += num;
    }
    return sum;
}

console.log(addAll(1, 2, 3));       // 6
console.log(addAll(1, 2, 3, 4, 5)); // 15
```

# 🔥 Final Summary Table

| Scenario | Parameters | Return | Use When |
|----------|------------|--------|----------|
| 1 | ✅ Yes | ✅ Yes | Need input, need result back |
| 2 | ✅ Yes | ❌ No | Need input, just perform a task |
| 3 | ❌ No | ✅ Yes | No input needed, need result back |
| 4 | ❌ No | ❌ No | No input, just perform a fixed task |

---

# 💡 Key Takeaway

- Use **return** when the caller needs the result to use further
- Skip **return** when the function just needs to perform an action
- Use **default parameters** to handle missing arguments safely
- Use **rest parameters** when number of arguments is not fixed
