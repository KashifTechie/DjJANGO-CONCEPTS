# JavaScript Data Types

---

# 🔹 1. Primitive Types

Stored by **value**, immutable, kept in **stack**.

| Type | Example |
|------|---------|
| Number | `42`, `3.14` |
| String | `"hello"` |
| Boolean | `true`, `false` |
| Undefined | `let x;` |
| Null | `let x = null;` |
| BigInt | `9999999999999999n` |
| Symbol | `Symbol("id")` |

```javascript
let a = 10;
let b = a;
b = 99;

console.log(a); // 10 → unchanged (independent copy)
console.log(b); // 99
```

---

# 🔹 2. Non-Primitive (Reference) Types

Stored by **reference**, mutable, kept in **heap**.

---

## 📌 2.1 Object

A collection of **key-value pairs**.

---

### ➤ Creating an Object

```javascript
let person = {
    name: "Kashif",
    age: 22,
    city: "Mumbai"
};
```

---

### ➤ Accessing Values

```javascript
// Dot notation
console.log(person.name);   // Kashif

// Bracket notation
console.log(person["age"]); // 22
```

---

### ➤ Setting / Updating Values

```javascript
person.name = "Ali";       // update existing key
person.country = "India";  // add new key

console.log(person.name);    // Ali
console.log(person.country); // India
```

---

### ➤ Deleting a Key

```javascript
delete person.city;

console.log(person.city); // undefined
```

---

### ➤ Checking if a Key Exists

```javascript
console.log("name" in person); // true
console.log("city" in person); // false
```

---

### ➤ Looping Over an Object

```javascript
for (let key in person) {
    console.log(key, person[key]);
}
// name Ali
// age 22
```

---

### ➤ Copying an Object

```javascript
// Shallow copy
let copy = { ...person };

// Deep copy
let deepCopy = JSON.parse(JSON.stringify(person));
```

---

### 🔧 Object Built-in Methods

| Method | What it does |
|--------|-------------|
| `Object.keys()` | Returns array of all keys |
| `Object.values()` | Returns array of all values |
| `Object.entries()` | Returns array of [key, value] pairs |
| `Object.assign()` | Copies properties into target object |
| `Object.freeze()` | Makes object immutable |
| `hasOwnProperty()` | Checks if key exists on object |

---

#### `Object.keys()`
```javascript
let person = { name: "Kashif", age: 22 };

console.log(Object.keys(person));
// ["name", "age"]
```

---

#### `Object.values()`
```javascript
console.log(Object.values(person));
// ["Kashif", 22]
```

---

#### `Object.entries()`
```javascript
console.log(Object.entries(person));
// [["name", "Kashif"], ["age", 22]]
```

---

#### `Object.assign()`
```javascript
let target = { a: 1 };
let source = { b: 2, c: 3 };

Object.assign(target, source);

console.log(target); // { a: 1, b: 2, c: 3 }
```

---

#### `Object.freeze()`
```javascript
let car = { brand: "Toyota" };
Object.freeze(car);

car.brand = "Honda"; // ❌ silently fails

console.log(car.brand); // Toyota
```

---

#### `hasOwnProperty()`
```javascript
let user = { name: "Ali" };

console.log(user.hasOwnProperty("name")); // true
console.log(user.hasOwnProperty("age"));  // false
```

---

## 📌 2.2 Array

An **ordered list** of values, 0-indexed.

---

### ➤ Creating an Array

```javascript
let fruits = ["apple", "banana", "mango"];
```

---

### ➤ Accessing Values

```javascript
console.log(fruits[0]); // apple
console.log(fruits[2]); // mango
```

---

### ➤ Setting / Updating Values

```javascript
fruits[1] = "grape";

console.log(fruits); // ["apple", "grape", "mango"]
```

---

### ➤ Checking Length

```javascript
console.log(fruits.length); // 3
```

---

### ➤ Looping Over an Array

```javascript
for (let fruit of fruits) {
    console.log(fruit);
}
// apple
// grape
// mango
```

---

### ➤ Copying an Array

```javascript
// Shallow copy
let copy = [...fruits];

// Deep copy
let deepCopy = JSON.parse(JSON.stringify(fruits));
```

---

### 🔧 Array Built-in Methods

| Method | What it does |
|--------|-------------|
| `push()` | Adds element to the **end** |
| `pop()` | Removes element from the **end** |
| `unshift()` | Adds element to the **beginning** |
| `shift()` | Removes element from the **beginning** |
| `map()` | Transforms each element, returns new array |
| `filter()` | Keeps elements that pass a condition |
| `reduce()` | Reduces array to a single value |
| `find()` | Returns first element that matches condition |
| `includes()` | Checks if a value exists |
| `indexOf()` | Returns index of a value |
| `slice()` | Returns portion of array (non-destructive) |
| `splice()` | Adds/removes elements (destructive) |
| `forEach()` | Loops through array (no return value) |
| `sort()` | Sorts array in place |
| `flat()` | Flattens nested arrays |

---

#### `push()`
```javascript
let arr = [1, 2, 3];

arr.push(4);

console.log(arr); // [1, 2, 3, 4]
```

---

#### `pop()`
```javascript
let arr = [1, 2, 3];

arr.pop();

console.log(arr); // [1, 2]
```

---

#### `unshift()`
```javascript
let arr = [1, 2, 3];

arr.unshift(0);

console.log(arr); // [0, 1, 2, 3]
```

---

#### `shift()`
```javascript
let arr = [1, 2, 3];

arr.shift();

console.log(arr); // [2, 3]
```

---

#### `map()`
```javascript
let numbers = [1, 2, 3, 4];

let doubled = numbers.map(n => n * 2);

console.log(doubled); // [2, 4, 6, 8]
console.log(numbers); // [1, 2, 3, 4] → unchanged
```

---

#### `filter()`
```javascript
let numbers = [1, 2, 3, 4, 5, 6];

let evens = numbers.filter(n => n % 2 === 0);

console.log(evens); // [2, 4, 6]
```

---

#### `reduce()`
```javascript
let numbers = [1, 2, 3, 4];

let sum = numbers.reduce((acc, curr) => acc + curr, 0);

console.log(sum); // 10
```

---

#### `find()`
```javascript
let users = [
    { id: 1, name: "Ali" },
    { id: 2, name: "Kashif" }
];

let user = users.find(u => u.id === 2);

console.log(user); // { id: 2, name: "Kashif" }
```

---

#### `includes()`
```javascript
let arr = [10, 20, 30];

console.log(arr.includes(20)); // true
console.log(arr.includes(99)); // false
```

---

#### `indexOf()`
```javascript
let arr = ["a", "b", "c"];

console.log(arr.indexOf("b")); // 1
console.log(arr.indexOf("z")); // -1 (not found)
```

---

#### `slice()`
```javascript
let arr = [1, 2, 3, 4, 5];

console.log(arr.slice(1, 3)); // [2, 3]
console.log(arr);             // [1, 2, 3, 4, 5] → unchanged
```

---

#### `splice()`
```javascript
let arr = [1, 2, 3, 4, 5];

arr.splice(2, 2); // start at index 2, remove 2 elements

console.log(arr); // [1, 2, 5]
```

---

#### `forEach()`
```javascript
let fruits = ["apple", "banana", "mango"];

fruits.forEach((fruit, index) => {
    console.log(index, fruit);
});
// 0 apple
// 1 banana
// 2 mango
```

---

#### `sort()`
```javascript
let names = ["Zara", "Ali", "Kashif"];
names.sort();
console.log(names); // ["Ali", "Kashif", "Zara"]

let nums = [10, 1, 5, 3];
nums.sort((a, b) => a - b); // ascending
console.log(nums); // [1, 3, 5, 10]
```

---

#### `flat()`
```javascript
let nested = [1, [2, 3], [4, [5, 6]]];

console.log(nested.flat());  // [1, 2, 3, 4, [5, 6]]
console.log(nested.flat(2)); // [1, 2, 3, 4, 5, 6]
```

## 📌 2.3 Map

Like an object but **keys can be any type** and insertion order is always preserved.

---

### ➤ Creating a Map

```javascript
let map = new Map();
```

---

### ➤ Setting Values

```javascript
map.set("name", "Kashif");
map.set(1, "one");
map.set(true, "yes");
```

---

### ➤ Accessing Values

```javascript
console.log(map.get("name")); // Kashif
console.log(map.get(1));      // one
console.log(map.get(true));   // yes
```

---

### ➤ Checking Size

```javascript
console.log(map.size); // 3
```

---

### ➤ Checking if Key Exists

```javascript
console.log(map.has("name")); // true
console.log(map.has("age"));  // false
```

---

### ➤ Deleting a Key

```javascript
map.delete("name");

console.log(map.has("name")); // false
```

---

### ➤ Looping Over a Map

```javascript
map.forEach((value, key) => {
    console.log(key, value);
});
// 1 one
// true yes
```

---

### ➤ Clearing All Entries

```javascript
map.clear();

console.log(map.size); // 0
```

---

### 🔧 Map Built-in Methods

| Method | What it does |
|--------|-------------|
| `set(key, value)` | Adds or updates a key-value pair |
| `get(key)` | Returns value for given key |
| `has(key)` | Checks if key exists |
| `delete(key)` | Removes a key-value pair |
| `clear()` | Removes all entries |
| `forEach()` | Loops through all entries |

> All methods shown with examples above ✅

---

## 📌 2.4 Set

A collection of **unique values only** — duplicates are automatically ignored.

---

### ➤ Creating a Set

```javascript
let set = new Set([1, 2, 2, 3, 3, 3]);

console.log(set); // Set {1, 2, 3} → duplicates removed
```

---

### ➤ Adding Values

```javascript
set.add(4);
set.add(2); // already exists → ignored

console.log(set); // Set {1, 2, 3, 4}
```

---

### ➤ Checking if Value Exists

```javascript
console.log(set.has(2)); // true
console.log(set.has(9)); // false
```

---

### ➤ Checking Size

```javascript
console.log(set.size); // 4
```

---

### ➤ Deleting a Value

```javascript
set.delete(1);

console.log(set); // Set {2, 3, 4}
```

---

### ➤ Looping Over a Set

```javascript
set.forEach(value => {
    console.log(value);
});
// 2
// 3
// 4
```

---

### ➤ Clearing All Values

```javascript
set.clear();

console.log(set.size); // 0
```

---

### 🔧 Set Built-in Methods

| Method | What it does |
|--------|-------------|
| `add(value)` | Adds a value (ignored if duplicate) |
| `has(value)` | Checks if value exists |
| `delete(value)` | Removes a value |
| `clear()` | Removes all values |
| `forEach()` | Loops through all values |

> All methods shown with examples above ✅

---

### ➤ Most Common Use Case — Remove Duplicates from Array

```javascript
let arr = [1, 1, 2, 2, 3, 3];

let unique = [...new Set(arr)];

console.log(unique); // [1, 2, 3]
```

# 🔥 Final Summary Table

| Type | Mutable | Keys | Order Preserved | Use When |
|------|---------|------|-----------------|----------|
| Object | ✅ | String / Symbol | ❌ | Named key-value storage |
| Array | ✅ | Numeric index | ✅ | Ordered list of values |
| Map | ✅ | Any type | ✅ | Keys of any type needed |
| Set | ✅ | — | ✅ | Unique values only |

---

# 💡 Key Takeaway

- **Object** → use for structured data with named keys
- **Array** → use for ordered lists
- **Map** → use when key needs to be a number, boolean, or object
- **Set** → use when you need to eliminate duplicates
